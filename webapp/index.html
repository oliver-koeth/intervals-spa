Settings

Data privacy

ℹ This app stores settings and optional intervals cache in your browser Local Storage. This implementation has no built-in analytics, tracking pixels, or telemetry. Data is sent only when you trigger API actions, and only to intervals.icu, Strava, and optionally your own local proxy endpoint (/api/*) when using Proxy mode. On page load, the chart library is also fetched from jsDelivr CDN. Use Reset all settings to remove stored values from this browser.

Note: this is a technical transparency summary, not a complete GDPR privacy notice.

Intervals.icu Connection

⚠ Enter your Athlete ID and API key to enable search. Find your athlete ID at the bottom of intervals.icu/settings and generate an API key under Account → API access.

Strava Connection

Stored locally in this browser only.

⚠ Enter your Strava Client ID and Client Secret, then connect OAuth to retrieve an access token with read + activity:read_all scope for segment search and HR streams. If Strava shows redirect_uri invalid, set OAuth Redirect URI to exactly the callback URL configured in your Strava app.

API Mode

ℹ Auto is recommended — it uses the local Python proxy when running on localhost and direct API access on GitHub Pages. Set to Proxy only if you are running server.py locally and want to force it. Set to Direct only for debugging.

HR Zone Model

ℹ Without a zone model the zone histogram uses a generic Z1–Z5 split. Load your sport settings from intervals.icu to get named zones with your actual LTHR thresholds and coloured HR stream bands.

Manual

This guide explains setup and usage from configuration to comparison. Click any screenshot thumbnail to open the full-size image.

1. Configuration first

Before searching, collect your connection data from Strava and Intervals.icu.

Strava

Open Strava API settings, click Edit, and set Authorization Callback Domain to oliver-koeth.github.io.

Then note: Client ID, Client Secret, Access Token, Refresh Token.

Intervals.icu

Open intervals.icu settings, scroll to Developer Settings, and note your Athlete ID and API Key.

2. Enter settings in Intervals SPA

In Settings, fill Intervals.icu and Strava fields, click Save, then Connect Strava. If needed, set OAuth Redirect URI to exactly the callback URL from your Strava app.

3. Search and save results

Use Search Intervals or Search Strava Segments, review preview results, then click Add all to intervals to store them in your browser.

4. Work with stored intervals

On the Intervals page, apply filters, select rows, and click Compare selected.

5. Compare in dashboard

In Compare, click a top-chart point to pin the lower charts (zone distribution and HR stream) to one interval.

Quick summary

  1. Get Strava and Intervals.icu credentials.
  2. Save credentials in Settings and connect Strava.
  3. Optionally load HR zones.
  4. Search, preview, and add results to Intervals.
  5. Filter, select, and compare intervals.
