Settings

Data privacy

ℹ This app stores settings and optional intervals cache in your browser Local Storage. This implementation has no built-in analytics, tracking pixels, or telemetry. Data is sent only when you trigger API actions, and only to intervals.icu, Strava, and optionally your own local proxy endpoint (/api/*) when using Proxy mode. On page load, the chart library is also fetched from jsDelivr CDN. Use Reset all settings to remove stored values from this browser.

Note: this is a technical transparency summary, not a complete GDPR privacy notice.

Intervals.icu Connection

⚠ Enter your Athlete ID and API key to enable search. Find your athlete ID at the bottom of intervals.icu/settings and generate an API key under Account → API access.

Strava Connection

Stored locally in this browser only.

⚠ Enter your Strava Client ID and Client Secret, then connect OAuth to retrieve an access token with read + activity:read_all scope for segment search and HR streams.

API Mode

ℹ Auto is recommended — it uses the local Python proxy when running on localhost and direct API access on GitHub Pages. Set to Proxy only if you are running server.py locally and want to force it. Set to Direct only for debugging.

HR Zone Model

ℹ Without a zone model the zone histogram uses a generic Z1–Z5 split. Load your sport settings from intervals.icu to get named zones with your actual LTHR thresholds and coloured HR stream bands.
